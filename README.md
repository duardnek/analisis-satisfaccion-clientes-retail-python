# Análisis de Satisfacción y Retención de Clientes – Retail

## Descripción

Este proyecto realiza un análisis exploratorio de datos (EDA) sobre más de **30,000 transacciones retail** con el objetivo de identificar patrones de satisfacción del cliente, segmentar riesgos de fuga y generar insights estratégicos para mejorar la retención.

El análisis se desarrolló utilizando Python y librerías de análisis de datos.

---

## Objetivos del análisis

- Identificar patrones de satisfacción del cliente.
- Detectar clientes con riesgo de abandono.
- Analizar el desempeño de productos, marcas y regiones.
- Generar insights para mejorar estrategias de retención.

---

## Herramientas utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
- Google Colab

---

## Dataset

El dataset contiene información de **30,062 transacciones** con variables como:

- Producto
- Categoría
- Marca
- Región
- Precio
- Cantidad
- Total de ventas
- Rating de satisfacción
- Tipo de venta

---

## Análisis realizado

- Limpieza y transformación de datos.
- Creación de variables analíticas:
  - Nivel de riesgo del cliente
  - Segmentación por rango de precio
  - Identificación de clientes con riesgo de fuga.
- Análisis de ventas por categoría, región y marca.
- Visualización de patrones mediante gráficos.

---

## Principales hallazgos

- La empresa presenta **alta eficiencia en ventas**, pero baja fidelización de clientes.
- Existe un gran segmento de **clientes neutrales con riesgo de migrar a la competencia**.
- Las categorías tecnológicas concentran la mayor parte de los ingresos.
- No se identificó un producto específico responsable de la insatisfacción, lo que sugiere problemas operativos generales.
